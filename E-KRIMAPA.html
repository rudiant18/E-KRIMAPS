⬡ Tentang Sistem

E-KRIMMAP adalah sistem informasi geografis (SIG) berbasis web untuk pemetaan daerah rawan kriminalitas di wilayah Kecamatan Cepu dan Kecamatan Padangan, Kabupaten Blora, Jawa Tengah.

Sistem ini dikembangkan sebagai tugas mata kuliah Sistem Informasi Geografis (SIG) dengan memanfaatkan teknologi peta terbuka (OpenStreetMap via Leaflet.js) dan penyimpanan data lokal tanpa biaya server.

Teknologi yang digunakan: HTML5, CSS3, JavaScript (Vanilla), Leaflet.js, OpenStreetMap, localStorage API.

Deploy gratis: Sistem ini dapat di-host langsung di GitHub Pages tanpa memerlukan database berbayar.

Koordinat wilayah: Kec. Cepu (−7.13°LS, 111.59°BT) · Kec. Padangan (−7.10°LS, 111.50°BT)
